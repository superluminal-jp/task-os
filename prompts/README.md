# prompts/

Claude Code のチャットに **そのまま貼れる短文** を置いています。スラッシュコマンド（`.claude/skills/`）と併用します。

| ファイル | 用途 |
| --- | --- |
| [daily-prompts.md](daily-prompts.md) | 朝・日中・夜・週次の呼び方（`/start-day`、`@daily-operator` など） |
| [setup-prompts.md](setup-prompts.md) | 初回セットアップ・棚卸し・既存メモの取り込み（`/new-task` と組み合わせ） |

運用の全体像は [docs/quickstart.md](../docs/quickstart.md) と [README.md](../README.md) を参照してください。
